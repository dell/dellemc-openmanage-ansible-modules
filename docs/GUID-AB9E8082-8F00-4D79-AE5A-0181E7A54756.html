Port breakout

ome_network_port_breakout
Synopsis
  • This module allows to automate the breaking out of IOMs in fabric mode into logical sub ports.
  • The port breakout operation is only supported in OpenManage Enterprise Modular.

Options

Table 1. ome_network_port_breakout
Parameter Required Default Choices Comments
hostname True NA NA OpenManage Enterprise Modular IP address or hostname
username True NA NA OpenManage Enterprise Modular username
password True NA NA OpenManage Enterprise Modular password
port False 443 NA OpenManage Enterprise Modular HTTPS port
target_port True NA NA The ID of the port in the switch to breakout. Enter the port ID in the format: Service tag:port. For example, 2HB7NX2:ethernet1/1/13.
breakout_type True NA NA
  • The preferred breakout type. For example, 4X10GE.
  • To revoke the default breakout configuration, enter 'HardwareDefault'.

Examples 

- name: Port breakout configuration
  dellemc.openmanage.ome_network_port_breakout:
    hostname: "192.168.0.1"
    username: "username"
    password: "password"
    target_port: "2HB7NX2:phy-port1/1/11"
    breakout_type: "1X40GE"
- name: Revoke the default breakout configuration       
  dellemc.openmanage.ome_network_port_breakout:
    hostname: "192.168.0.1"
    username: "username"
    password: "password"
    target_port: "2HB7NX2:phy-port1/1/11"
    breakout_type: "HardwareDefault
